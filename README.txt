Сервер запускается на порту, который указан в environment variables.

Клиентов-плагинов может быть хоть сколько. Каждый из них при подключении отправляет серверу указание на то, как часто им надо отправлять сообщения (есть вариант рандома).

Под каждого клиента выделен свой поток, отсчет для каждого клиента идет по-своему.

Запуск получения фраз по команде "phrase". Фразы выбираются случайно из файла.

Полученная фраза выводится в игру.
